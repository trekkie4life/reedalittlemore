FB Cleanr

The problem: Facebook's latest update clutters your screen with unwanted ads

Originally built for personal use because I was tired of having such a busy page when I went to Facebook, I have released the extension for free to all. I'm sure I'm not the only person who doesn't like ads or clutter and only wants to only have relevant information on the screen.

Because Facebook disables jQuery and also BFCache'ing, it was difficult to find a way to always maintain the new 'ad-free' state invoked by the extension. The way around this was to have Chrome's API monitor the Facebook tab. If the tab went inactive, then became active again, the API would fire the javascript to re-remove the ads.
