DBC Overflow

The problem: There is no single unified place to learn the ins and outs of DBC life

DBC Overflow provides a single location for all questions/concerns and answers about DBC life/studies/preparation for potential applicants of DBC or current Phase 0 DBC students. Both questions and answers can be voted on, but only the question asker can select a "best" answer. You can also search for questions.

In true geek fashion, the favorited question gets a Yoda icon by it to set it apart.

Stack: Ruby on Rails, PostgreSQL, JavaScript, AJAX
